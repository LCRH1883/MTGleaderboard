# Versioning

## Version Name Format

Version names use the format `vMM.mm.ppp`.

- `MM` = major version (2 digits, zero-padded)
- `mm` = minor version (2 digits, zero-padded)
- `ppp` = patch version (3 digits, zero-padded)

Example: `v1.3.1` is written as `v01.03.001`.

## Version Number Format

Version numbers are the digits only: `MMmmppp`.

Example: `v1.3.1` -> `0103001`.

## Current Version

- Version name: `v00.01.001`
- Version number: `0001001`
